🔧 Установка
1. Клонирование репозитория
bash
git clone https://github.com/1skanderF/Hakaton_2026_YF.git
cd Hakaton_2026_YF
2. Установка зависимостей
bash
pip install ultralytics opencv-python numpy matplotlib pyTelegramBotAPI python-dotenv
Или используйте requirements.txt (создайте файл):

bash
pip install -r requirements.txt

⚙️ Настройка
1. Получение токена Telegram бота
Найдите в Telegram @BotFather

Отправьте команду /newbot

Введите имя бота (например, Plant Analyzer Bot)

Введите username (должен заканчиваться на bot, например my_plant_analyzer_bot)

Скопируйте полученный токен (выглядит как 7234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw)

2. Создание файла .env
Создайте файл .env в корневой папке проекта:

text
BOT_TOKEN=7234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
ВАЖНО: Замените токен на свой!

3. Проверка моделей
Убедитесь, что файлы моделей находятся в папке models/:

models/argula_best.pt - для анализа рукколы

models/wheat_best.pt - для анализа пшеницы

🔧 Устранение проблем
Ошибка "A request to the Telegram API was unsuccessful"
Причина: Неправильный или отсутствующий токен бота.

Решение:

Проверьте файл .env

Убедитесь, что токен правильный

Перезапустите бота

Ошибка "No module named 'ultralytics'"
Причина: Не установлены зависимости.

Решение:

bash
pip install ultralytics
Ошибка "File not found" при анализе
Причина: Неправильный путь к изображению.

Решение: Используйте абсолютный путь:

python
image_path = r"D:\полный\путь\к\файлу.jpg"
Файлы моделей слишком большие (>50 MB)
Рекомендация: Используйте Git LFS:

bash
git lfs install
git lfs track "models/*.pt"
git add .gitattributes
git commit -m "Add LFS for models"
